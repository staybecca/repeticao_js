Estruturas de repetição

Existem três tipos de estruturas de repetição de estruturas de repetição: for, while e do while,(algumas funcionam melhor que outras em casos especificos). Elas são importantes para automatizar tarefas repetitivas, reduzir a quantidade de código e melhorar a eficiência do programa mostrarei alguns exemplos praticos de cada


Diferenças entre elas

for é usado quando sabemos o número de iterações antecipadamente. Possui três partes: inicialização, condição e incremento/decremento.
while executa enquanto a condição especificada for verdadeira. A condição é verificada antes de cada iteração.
do while é similar ao while, mas a condição é verificada após a execução do bloco de código, garantindo que o código seja executado pelo menos uma vez.



for:

É otima quando se sabe o numero de vezes que voce quer repetir um bloco de codigo


Sintaxe:

bloco de codigo a ser executado enquanto a condição for verdadeira

for (inicialização; condição; expressão final){
//codigo que sera repetido }

while:

É otima quando se quer repetir um bloco de codigo enquanto uma condição é verdadeira


Sintaxe:

bloco de codigo a ser executado repetidamente enquanto uma condição especificada for verdadeira

while (condição){
//codigo que sera repetido }

do while:

É parecida com while, porem garante que o bloco sejaa executado pelo menos uma vez


Sintaxe:

bloco de codigo dentro do "do" é executado pelo menos uma vez antes da verificação da condição, em seguida a condição é verificada se for verdadeira, o bloco é executado de novo, senão o loop acaba

do {
//codigo que sera repetido
} while (condição);
